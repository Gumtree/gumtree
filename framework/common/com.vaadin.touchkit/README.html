Vaadin TouchKit 2.1.0

Vaadin TouchKit is an add-on component for Vaadin to aid creating applications for touch based devices like smartphones and tablets.

Vaadin TouchKit is a dual licensed product. See licensing.txt file for licensing details about the package you are using.

Manual

TouchKit manual is included as a chapter in the Book of Vaadin. The latest version can be found online at:
https://vaadin.com/book

Tutorial

In addition to the reference manual there is a more tutorial like article available how to create a full featured mobile application with Vaadin TouchKit. The reader is expected to have basic knowledge of Vaadin development.
