Safety PLC in SICS

by Douglas Clowes — last modified 2007-03-09 11:51

The lights on the Safety PLC panel in the instrument cabin can be read by SICS

The Safety PLC is represented in SICS by an object called "plc". Each of the lamps on the Safety PLC panel in the instrument cabin, except the lamp test button, are available through the plc object.


To list all the states of the lamps:

plc list
plc.Key = enabled
plc.Secondary = opened
plc.Tertiary = opened
plc.MotionControl = enabled
plc.Access = locked
plc.DC = false
plc.Exit = false
plc.Trip = false
plc.Fault = false
plc.Operate = false
plc.Relay = true
plc.Ready = true

Each field can be acquired individually by entering its name as the only parameter:

plc ready
plc.Ready = true

The fields and their meanings are:

  • Key - Secondary Shutter Key Administrative Control Enabled/Disabled.
  • Secondary - Secondary Shutter Opened/Closed.
  • Tertiary - Sample Shutter Opened/Closed.
  • MotionControl - Motion Control Enabled/Disabled.
  • Access - Instrument Area Access Locked/Unlocked.
  • DC - DC Supplies OK.
  • Exit - Exit In Progress.
  • Trip - Safety Interlock System Tripped.
  • Fault - Safety Interlock Malfunction.
  • Operate - Sample Shutter Operate.
  • Relay - Secondary Shutter Interfacing Relay Enabled
  • Ready - Instrument Ready.

The values for the fields "Key" through "Access" may also take the values "unknown(low)" if both lights are off, or "invalid(high)" if both lights are on.
