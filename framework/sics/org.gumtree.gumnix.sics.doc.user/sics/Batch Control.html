Batch Control

by Ferdi Franceschini — last modified 2007-01-18 11:34

You can execute a scripted sequence of commands using the SICS batch control mechanism.  The scripts must be stored in the /usr/local/sics/server directory.

A batch script is simply a Tcl script interspersed with SICS commands.  A script can be created remotely (via the GumTree client) or you could simply create a script with an editor and store it on the SICS host computer to run later.

Once you have created a script you can run it with the exe command.

eg

exe my_script
For more notes on creating batch files see  Batch Control
